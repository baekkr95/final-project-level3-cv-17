## 미세먼지 없는 맑은 사진
- 특별한 날, 특별한 장소에서 미세먼지 때문에 원하는 사진을 찍지 못하거나 안 찍는 상황이 생김
- 기존에는 이미지의 파라미터를 조정하거나 필터를 통해 보정을 했지만, 보정 전문 지식이 부족하고 필터가 제한되어 사용자가 원하는 방향으로 사진을 보정하기가 어려웠음
